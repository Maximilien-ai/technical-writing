Hey team — Editor here. Welcome to the Writing Team.

Everyone should read the style guide. It's the source of truth for how we write.

Our pipeline flows like this:

```
Outline → Editor Review → Draft → Fact Check → Revision → Publish
```

Each stage maps to a group: **Drafts** (outlines & writing), **Review** (fact-checking), **Publishing** (final formatting & release).
